What is Web 2.0 & Web 3.0?

"Web 2.0 and Web 3.0 represent successive, advanced iterations of the original Web 1.0 of the 1990s and early 2000s. Web 2.0 is the current version of the web with which we are all familiar, while Web 3.0 represents its next phase, which will be decentralized, open, and of greater utility." - Investopedia

With the intriduction of smartphones, mobile internet access, and social networks Web 2.0 has become the most dominate version of theweb as of recent years
